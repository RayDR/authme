# AuthMe - FastAPI Authentication System

This project is designed to handle authentication for all applications and APIs using OAuth2 with credentials. Built with FastAPI, the system is scalable and intended to be a future open-source project on GitHub.

## Setup

1. Clone the repository
2. Install dependencies using pip: `pip install -r requirements.txt`
3. Run the application: `uvicorn main:app --reload`

## Branching Strategy

We follow the GitFlow branching model. Please make sure to work on feature branches and follow the pull request process.
